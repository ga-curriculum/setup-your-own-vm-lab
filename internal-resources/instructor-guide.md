<h1>
  <span class="headline">Setup Your Own VM Lab</span>
  <span class="subhead">Instructor Guide</span>
</h1>

This lab is intended to be delivered in two parts:

- In **part 1**, students will install VirtualBox on their machines. They will also create a virtual machine running Ubuntu.
- In **part 2**, students will create a virtual machine running Windows, unless they are using a Mac with an Apple Silicon chip, in which case they will create a virtual machine running CentOS.
